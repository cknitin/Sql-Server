# Isolation Level in Sql Server

## Types of Isolation level

### 1. Read Uncommited
### 2. Read commited
### 3. Read Repeatable read
### 4. Snapshot
### 5. Serializable 


### Example

    SET TRANSACTION ISOLATION LEVEL
    { READ UNCOMMITTED
    | READ COMMITTED
    | REPEATABLE READ
    | SNAPSHOT
    | SERIALIZABLE
    }
